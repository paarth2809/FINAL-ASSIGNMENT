HTML tags

  1. heading tag: it is used to display the heading on web pages. There are different heading of different sizes from h1 to h6. h1 is the biggest size heading and h6 is smallest size heading.
  2. paragraph tag: it is used to make a paragraph that will be displayed on web page.
  3. break tag: it is used to break the current line and go to next line. Content will start displaying from next line.
  4. horizontal line tag: it is used to display a horizonatl line to our web page.

    5. There are many more tags like div,span,table etc.
